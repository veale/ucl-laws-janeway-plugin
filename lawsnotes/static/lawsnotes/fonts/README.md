# Bundled fonts

Newsreader, by Production Type, distributed under the SIL Open Font License
1.1. Source: https://github.com/productiontype/Newsreader

Two variable woff2 files are bundled, upright and italic, each carrying the
full 200–800 weight range and an optical-size axis. A single `@font-face` per
style therefore covers every weight the stylesheet uses.

The OFL permits bundling and redistribution, including in commercial work,
provided the font itself is not sold on its own and the licence travels with
it. Retain this notice if you redistribute the plugin.
